Krusader is an advanced twin panel (commander style) file manager for KDE Plasma and other desktops in the *nix world, similar to Midnight or Total Commander. It provides all the file management features you could possibly want. Plus:

Features

Extensive archive handling

Easily search information and access it faster while ensuring integrity for legal value preservation at the same time! Easily manage privacy data and get rid of legacy applications.

Mounted filesystem support

A mounting filesystem is tolerant to being relocated, can be load balanced, along with a whole host of other advantages and simultaneously is a filesystem which incorporates scalability and reliability enhancements for supporting large filesystem.

File transfers with (S)FTP

Keep your connection secured, while uninformed TCP/IP ports can be redirected through the encrypted channel in both directions.

Advanced search module

Integrate criteria for selecting information, including geographic, linguistic or file format criteria with Krusader!
